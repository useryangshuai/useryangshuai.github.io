标识符、关键字、数据类型

java中的注释

写在java源码中，在java中起到提示作用，不进行编译执行。

java中的注释有三种：

  1. 单行注释 ： //注释
  2. 多行注释 : /*注释可以换行*/
    1. 注释思路，或者程序逻辑，帮助后期的理解和维护
  3. 文档注释 : /**文档注释*/
    1. 常用的javadoc标记
    2. @author：作者
    3. @version：版本
    4. @docroot：表示产生文档的根路径
    5. @deprecated：不推荐使用的方法
    6. @param：方法的参数类型
    7. @return：方法的返回类型
    8. @see：”参见”，用于指定参考的内容
    9. @exception：抛出的异常
    10. @throws：抛出的异常，和exception同义

分隔符

  1. 一条语句是以分号（;）结尾的一行代码
  2. 一个语句块是以一对花括号（{}）为边界的语句的集合
  3. Java中允许有任意多的空格 ，包括换行,以及;

标识符

给代码起的名字

  • 类名
  • 方法名
  • 变量名
  • 常量名

标识符的组成

  • 标识符必须有 数字、字母（中文）、_、$ 组成
  • 标识符不能以数字开头
  • 标识符不能java保留字
  • java标识符是大小写敏感的
  • 表示符必须要有语义化（使用英文单词）
  • 大驼峰命名法（AaaaBbbbCcccc）
    • 类名
  • 小驼峰命名法 （aaaaBbbbCccc）
    • 方法名
    • 变量名/属性名
  • （AAAA_BBBB）
    • 用于定义常亮

Java关键字/保留字

  • Java中一些赋以特定的含义、并用做专门用途的单词称为关键字（keyword）
  • 所有Java关键字都是小写的，TURE、FALSE、NULL等都不是Java关键字 ；
  • goto和const 虽然从未被使用，但也作为Java关键字保留；
  • Java中一共有48+2个关键字，如下表所示

case catch char class const double

while void throw throws transient try
volatile enum

数据类型

  1. Java是一门强类型语言。也就是说，所有的变量都必须显式声明类型。

  2. Java的数据类型分为两大类：原始类型（primitive type，也称为简单类型）和引用类型（reference type）
